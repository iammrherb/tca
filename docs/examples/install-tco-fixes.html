Installation Instructions

To install all enhancements to the TCO Analyzer, add the following script tag just before the closing </body> tag in your HTML:

<script src="install-fixes.js"></script>

You can also use this one-liner to insert the script tag:

sed -i 's/<\/body>/<script src="install-fixes.js"><\/script>\n<\/body>/g' index.html
